City to Lat & Lng

A free and easy to use API.

Learn More

About

This API accepts a city and state and will return a JSON file containing the Latitude and Longitude coordinates.

It is built with Node/Express and deployed on Heroku.
Visit us on GitHub: city-to-lat-lng

Try It Now!

Usage


$.getJSON( "https://mrrockabilly.herokuapp.com/cityGeoCode?city=Los+Angeles&state=CA")
  .done(function( json ) {
    console.log(json);
  })
  .fail(function( jqxhr, textStatus, error ) {
    var err = textStatus + ", " + error;
    console.log( "Request Failed: " + err );
});
